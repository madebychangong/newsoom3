# 원고 수정 패턴 분석 (15개 실제 사례)

## 📊 발견된 주요 패턴

### 1. 특수 변환 규칙 (필수)
| 패턴 | 빈도 | 설명 |
|------|------|------|
| `네요` → `내요` | **80% (12/15)** | 가장 많이 적용되는 규칙 |
| `하더라` → `하더 라` | 26.7% (4/15) | 띄어쓰기 추가 |

**예시:**
```
수정 전: 마음이 너무 아프네요
수정 후: 마음이 너무 아프내요
```

### 2. 특수문자 사용 (서브키워드 확보)
- **사용 빈도: 46.7% (7/15)**
- 서브키워드가 부족할 때 특수문자로 보충

**가장 많이 사용된 특수문자:**
1. `;;` (7회) - 최선호
2. `..` (6회)
3. `!!` (6회)
4. `~~` (4회)
5. `^^` (4회)

**사용 패턴:**
```
"고민이 많아요 ;;"
"정말 걱정이에요 .."
"감사하겠습니다 !!"
"너무 힘들어요 ~~"
"좋은 정보 감사해요 ^^"
```

**특징:**
- 문장 끝에 자연스럽게 추가
- 감정을 표현하는 위치에 사용
- 앞뒤로 띄어쓰기 필수: ` ;; `

### 3. 키워드 배치 최적화

#### 첫 문단 키워드 감소 (33.3%)
- 첫 문단에 키워드가 과다하면 **제거**
- 목표: 정확히 2개 유지

#### 전체 키워드 증가 (60%)
- 나머지 문단에 키워드 **추가**
- 목표 범위(N ~ N+1) 맞추기

### 4. 문장 구조 개선 (80%)
- **짧은 문장** 선호
- 긴 문장 → 여러 개의 짧은 문장으로 분리
- 가독성 향상

**예시:**
```
수정 전: 아이가 또래보다 작아서 줄 맨 앞에 서는데 마음이 아프네요.
수정 후: 아이가 또래보다 작아요. 줄 맨 앞에 섭니다. 마음이 아프내요.
```

### 5. 글자수 유지
- 평균 변화: **+3자** (565 → 568)
- 대부분 300~900자 범위 유지
- 큰 변화 없이 규칙만 충족

---

## 🎯 수정 전략 정리

### 우선순위
1. **특수 변환 적용** (네요→내요, 하더라→하더 라)
2. **키워드 개수 조정** (첫 문단 2개, 나머지 목표±1)
3. **서브키워드 확보** (부족하면 특수문자 추가)
4. **문장 짧게 다듬기** (가독성)

### 특수문자 사용 전략
- 서브키워드 부족 시 **우선 사용**: `;;`, `..`, `!!`
- 감정 표현 문장 끝에 자연스럽게 삽입
- 다양한 패턴 사용 (각각 2회씩 등장해야 서브키워드로 카운팅)

---

## 📌 핵심 인사이트

1. **"내요" 변환은 거의 필수** - 80%에서 사용
2. **특수문자는 선택적** - 46.7%만 사용 (부족할 때만)
3. **`;;` 를 가장 선호** - 다른 특수문자보다 2배 많이 사용
4. **문장은 짧고 간결하게** - 80%에서 문장 수 증가
5. **글자수는 거의 변하지 않음** - 평균 +3자만 변화

## 🔍 실제 케이스 분석

### Case 1: 키탑 성장클리닉
- 글자수: 600 → 601 (+1)
- 통키워드: 2 → 5 (+3)
- 특수 변환: "아프네요" → "아프내요"
- 전략: 나머지 문단에 키워드 추가

### Case 2: 초3 여아키
- 글자수: 549 → 556 (+7)
- 통키워드: 5 → 6 (+1)
- 특수 변환: "아프네요" → "아프내요"
- 전략: 최소한의 키워드만 추가

### Case 3: 아이키플러스
- 글자수: 541 → 200 (-341) ⚠️ 대폭 감소
- 통키워드: 6 → 2 (-4)
- 특수문자: `!!` 사용 확인
- 전략: 내용 대폭 축약 (특이 케이스)
