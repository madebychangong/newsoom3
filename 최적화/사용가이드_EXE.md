# 📱 블로그 검색 최적화 시스템 사용 가이드

검색 노출을 위한 키워드 띄어쓰기 및 금칙어 최적화!

---

## 🚀 빠른 시작

### 1. 프로그램 실행

`블로그SEO최적화.exe` 더블클릭

### 2. 파일 선택

- **"📁 파일 선택"** 버튼 클릭
- Excel 파일 (.xlsx) 또는 TXT 파일 (.txt) 선택

### 3. 설정 (선택사항)

- **키워드:** TXT 파일은 자동 추출, 수동 입력 가능
- **브랜드명:** 제품 브랜드 입력 (선택, 해시태그에 포함됨)

### 4. 최적화 실행

**"🚀 최적화 시작"** 버튼 클릭

### 5. 결과 확인

- 진행 상황이 로그 창에 표시됩니다
- 완료 후 **"📂 결과 폴더 열기"** 클릭

---

## 📂 파일 형식

### Excel 파일 (.xlsx)

**필수 컬럼:**
- 키워드
- 원고

**선택 컬럼:**
- 브랜드

**결과:** 같은 폴더에 `원본파일_검색최적화.xlsx` 생성

### TXT 파일 (.txt)

**형식:**
```
# 키워드 관련 제목

원고 내용...
```

**결과:** 같은 폴더에 `원본파일_최적화.txt` 생성

---

## 🎯 최적화 내용

### 검색 노출 최적화

1. ✅ **# 제목 삭제:** 첫 줄 제목 자동 제거
2. ✅ **키워드+조사 제거:** 키워드를/가/이/에 → 자연스럽게 수정
3. ✅ **키워드 출현 감소:** 5-6회 → 2-3회 (스팸 방지)
4. ✅ **금칙어 치환:** 네이버 금칙어 자동 치환
5. ✅ **AI 표현 제거:** 부자연스러운 AI 문체 제거
6. ✅ **해시태그 생성:** 8-10개 자동 생성

### 키워드 띄어쓰기 규칙

```
❌ 고관절영양제에 대해  → ✅ (문장 삭제)
❌ 고관절영양제를        → ✅ 고관절영양제
❌ 고관절영양제가        → ✅ 고관절영양제 먹으면
❌ 고관절영양제라는      → ✅ 고관절영양제 라는 (띄어쓰기)
```

## 📊 결과 파일 내용

### TXT 출력

```
================================================================================
블로그 원고 검색 최적화 결과
================================================================================

📊 최적화 정보
--------------------------------------------------------------------------------
키워드: 팔꿈치 쿠션 보호대
글자수: 1852자 (+0자)
키워드 출현: 2회

🔧 변경 사항
--------------------------------------------------------------------------------
✅ # 제목 삭제
✅ 키워드+조사 제거 (6회)
✅ 키워드 출현 감소 → 2회
✅ 금칙어 3개 치환
✅ AI 표현 2개 수정

🏷️ 추천 해시태그
--------------------------------------------------------------------------------
#팔꿈치쿠션보호대 #팔꿈치 #쿠션 #보호대 #초록입환 #건강정보 #일상

📌 제목
--------------------------------------------------------------------------------
팔꿈치 쿠션 보호대 정보 찾아봤어요

================================================================================
📝 최적화된 원고
================================================================================

(최적화된 내용...)
```

### Excel 출력

**추가되는 컬럼:**
- 최적화_원고
- 키워드_출현
- 변경사항
- 추천_해시태그

## 🔧 고급 사용법

### 여러 파일 일괄 처리 (Excel)

Excel 파일에 여러 행의 원고를 입력하면 자동으로 모두 처리됩니다!

### 금칙어 커스터마이징

`금칙어 수정사항 모음.txt` 파일을 메모장으로 열어 수정:

```
금칙어
대체어1 / 대체어2 / 대체어3

효과
약효

병원
병의원 / 센터 / 클리닉
```

**형식:**
- 금칙어 (첫 줄)
- 대체어들 (둘째 줄, `/` 또는 `,`로 구분)
- 빈 줄
- 다음 금칙어...

## ⚠️ 주의사항

### 필수 파일

**중요:** `금칙어 수정사항 모음.txt` 파일을 실행 파일과 **같은 폴더**에 보관하세요!

```
📁 내폴더/
  ├── 블로그SEO최적화.exe        ← 실행 파일
  └── 금칙어 수정사항 모음.txt    ← 필수!
```

### 원고 길이

- **원본 길이 유지:** 글자수를 늘리거나 줄이지 않습니다
- **키워드 최적화만 수행:** 검색 노출을 위한 키워드 처리에 집중

## ❓ FAQ

### Q1: 프로그램이 실행되지 않아요

**A1:** Windows Defender 경고가 뜰 수 있습니다.
- "추가 정보" 클릭 → "실행" 선택

**A2:** Visual C++ 재배포 패키지 필요
- https://aka.ms/vs/17/release/vc_redist.x64.exe 설치

### Q2: 금칙어 파일을 찾을 수 없다고 나와요

**A:** 실행 파일과 같은 폴더에 `금칙어 수정사항 모음.txt`를 복사하세요.

### Q3: Excel 파일이 열리지 않아요

**A:** Excel 파일이 다른 프로그램에서 열려있으면 저장할 수 없습니다. 닫고 다시 시도하세요.

### Q4: 키워드가 너무 많이 제거됐어요

**A:** 정상입니다. 검색 스팸 방지를 위해 키워드 출현을 2-3회로 제한합니다. 자연스러운 검색 노출에 더 효과적입니다.

### Q5: 키워드가 자동으로 잘 안 잡혀요

**A:** TXT 파일 첫 줄을 `# 키워드 ...` 형식으로 작성하거나, GUI에서 수동으로 입력하세요.
