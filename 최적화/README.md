# 블로그 검색 최적화 프로그램

네이버 블로그 원고를 검색 노출에 최적화하는 프로그램입니다.

## ✨ 주요 기능

### 키워드 최적화
- ✅ 키워드 띄어쓰기 자동 수정
- ✅ 키워드+조사 제거 (를/을/가/이/에/의/는/은)
- ✅ 키워드 출현 빈도 최적화 (5-6회 → 2-3회)
- ✅ # 제목 자동 제거

### 금칙어 치환
- ✅ 네이버 금칙어 자동 치환
- ✅ AI 느낌 나는 표현 제거
- ✅ 자연스러운 구어체로 변환

### 엑셀 일괄 처리
- ✅ 여러 원고를 한 번에 처리
- ✅ 키워드, 브랜드별 자동 최적화
- ✅ 결과를 새 엑셀 파일로 저장

## 📦 프로그램 구성

### GUI 프로그램 (EXE)
```
블로그SEO최적화.exe
```
- Windows에서 바로 실행 가능
- 엑셀 파일을 드래그 앤 드롭으로 처리
- 실시간 진행 상황 표시

### 필요한 파일
```
블로그SEO최적화.exe       # 실행 파일
금칙어 수정사항 모음.txt    # 금칙어 목록 (같은 폴더에 있어야 함)
```

## 🚀 사용 방법

### 1. EXE 실행
1. `블로그SEO최적화.exe` 더블클릭
2. "파일 선택" 버튼 클릭 또는 엑셀 파일 드래그
3. "최적화 시작" 버튼 클릭
4. 처리 완료되면 `_검색최적화.xlsx` 파일 생성

### 2. 엑셀 파일 형식
필수 열:
- `키워드`: 최적화할 키워드
- `원고`: 최적화할 원고 내용
- `브랜드`: (선택) 브랜드명

출력 열:
- `최적화_원고`: 최적화된 원고
- `키워드_출현`: 키워드 출현 횟수
- `변경사항`: 변경 내용 요약
- `추천_해시태그`: 추천 해시태그 목록

## 🔧 개발자용

### 빌드 방법
```bash
# Windows
build.bat

# Linux/Mac
./build.sh
```

### 테스트 방법
```bash
# 빠른 테스트 (첫 번째 원고만)
python3 quick_test.py

# 대화형 테스트
python3 test_optimizer.py
```

### 필요한 패키지
```bash
pip install -r requirements.txt
```

## ⚙️ 최적화 규칙

### 키워드 띄어쓰기
```
❌ 고관절영양제에 대해  → ✅ (문장 삭제)
❌ 고관절영양제를        → ✅ 고관절영양제
❌ 고관절영양제가        → ✅ 고관절영양제 먹으면
❌ 고관절영양제라는      → ✅ 고관절영양제 라는 (띄어쓰기)
```

### 금칙어 치환
```
병원 → 센터, 클리닉
효과 → 약효
시술 → 치료
의사 → 담당의, 선생님
```

### AI 표현 제거
```
정말 고민이 많습니다 → 정말 고민되네요
절로 나오 → 자연스럽게 나오
고생하고 있는 → 힘들어하는
```
